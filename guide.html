🏇 競馬予想ページの読み方と使い方

URL: https://coca448.github.io/keiba-share/ — 日付を選ぶとその開催の全レースが見られます(スマホ対応)。

このページには2つのAIと手動ツールが同居しています。まずそこだけ押さえてください:
  • R系(RaceReader) … 過去10年・約50万走のデータから「どの馬が強いか」を学習したAI
  • P系(PaceMaker) … 出走メンバーから「レースがどう流れるか(隊列・ペース・上がり)」をシミュレートするAI
  • 手動 … 展開図の駒を自分で動かしたり、馬場の偏りを自分で設定する部分
RとPは見ているものが違うので、意見が割れます(74%のレースで本命が違う)。割れた時に統計的に当たりやすいのはR側です。Pは「展開がハマればこの馬」という第二意見として読んでください。

画面の構成(上から)

1. ヘッダ

  • ペース想定(スロー/ミドル/ハイ) … 「行きたい馬」が何頭いるかから機械的に判定。逃げたい馬がいなければスロー、2頭以上ならハイになりやすい
  • 展開図の信頼度(高/中/割引) … 逃げ馬不在のレースは隊列が読みやすく(信頼度高)、逃げたい馬が複数いると隊列が流動的(割引)。実測に基づく自己申告です

2. 展開想定図(俯瞰の盤面)

コースを上から見た図に、4コーナー時点の予想配置で駒(実寸比の楕円)を置いています。

  • 駒の位置 … 縦(前後)=テンの速さと過去の位置取り履歴から予測。横(内外)=基本は枠なり(内枠が内)。あくまで平均的な想定=叩き台です
  • 角タブ(1角/2角/3角/4角) … コーナーごとの想定を切替表示。注意書きの通り、コマ間の「動き」は型の平均像で、実際のレースはこの絵の約2倍動きます
  • ⤴マーク … 捲り型(道中で大きく位置を上げる型)。この印の馬が次走で実際に捲る率は約8.7%(無印馬の2.5倍)
  • 動マーク … 機動力型(位置を動かせる型)。次走で位置が動く率55%(無印は24%)。⤴や動の駒は「絵の位置から動く可能性が高い」と読んでください
  • 駒のタップ/ホバー … 距離ロス(外を回ると何m損か)・壁リスク・想定上がりを表示。距離ロスと壁は「参考」——オッズに織り込み済みの情報で、これ単体で儲かるものではありません
  • 駒のドラッグ … 自分の読みで配置を修正できます(前に出す・外に出す)。動かすとゴールタイム計算が連動して変わります
  • 帯のタップ … 馬場の偏り(TB)の手動設定。「内が伸びる/死んでいる」を自分の目で入れる欄です(機械は当日の馬場を知りません)

3. 馬表(全頭・横スクロール、印と馬名は固定)

列意味数字の出所
印(一番左)自分で打つ印。タップで◎○▲△✕を切替あなた
R順/R印強さの序列(AIの本命)RaceReaderの評価
勝率「32%」なら、過去に同じ評価を受けた馬が実際に約32%勝っている、という意味の校正済みの数字同上(実測で校正)
信(A/B/C)AIの自信度。実測でA約66%・B約61%・C約50%(本命の複勝率)と確認済み。Cは「読めていない」の正直な申告同上
着順レンジ予測の幅。広い馬は成績にムラがある同上
ペ適(◎○▲×)今日の想定ペース×その馬の脚質の相性。6年23万頭の実測マップ(「前有利は常に、逃げ馬不在なら特に」)実測統計
好走パターン/根拠AIがその評価をした理由の文章RaceReaderの理由生成
P順展開シミュレーションの着順序列。「4角の位置の貯金+直線で使える脚」を秒に換算して合計した順番。印ではありません(Rと割れたらRの方が当たる)PaceMaker
想定上がり「今日のペース・位置なら、この馬は直線で何秒の脚を使えるか」の予測。過去の上がりの平均ではなく、ペースを考慮した予測なのがミソ同上
4角位置4コーナーの予想位置(盤面の駒と同じ)同上
4角/上がり/実効/ゴール決着タイムのシミュレーション。ゴール=4角通過タイム+上がり+補正(競られ・馬場)。上がりの初期値は「想定上がり」で、スライダーで手動調整も可能シミュ(手動調整可)
実着レース後に表示される実際の着順結果

4. 答え合わせ(レース後)

「答え合わせ」トグルをONにすると、実際の4コーナー配置(ゴースト)が予想の盤面に重なり、自分の印/R本命/P順1位それぞれの結果が並びます。予想のどこが合ってどこがズレたかを、3つの視点で振り返れます。

使い方(1レースの流れ)

1. レースを選ぶ → ヘッダで地形を掴む(ペース想定と信頼度)
2. 展開図を見る → 変だと思う駒をドラッグで直す(⤴・動の駒は特に「動くかも」と疑う)
3. 当日の馬場を見て帯タップでTBを設定(内有利/外有利)
4. 馬表でRとPを見比べる → 「Rの評価は高いが展開不利」「評価は2番手だが展開最高」のズレを探す
5. 一番左の列で自分の印を打つ → 「この予想を保存」
6. レース後、答え合わせで振り返る

注意書き

  • 数字には信頼度の階層があります: R系の勝率・信頼度(実測で校正済み) > P順・想定上がり(検証済みの予測) > 距離ロス・壁(参考値) > TB(手動入力)。画面の注記(「参考」「手動」)はこの区別です
  • 展開図の初期配置は平均像です。当日のゲート・出遅れ・騎手の判断は入っていません——そこを直すのが使い手の仕事です
  • 保存した印はお使いの端末・ブラウザの中だけに保存されます(他の人には見えません)
  • 予想は参考情報です。馬券の購入は自己責任で
